Frequently Asked Question

Need some customization in this app, whom to contact?
Please drop email at sales@ksolves.com or raise a ticket through Odoo store itself.
Do you provide any free support?
Yes, we do provide free support for 90 days for any queries or any bug/issue fixing.
Do I have to buy the module for each version?
Every module bought for version 12.0 and prior gives you access to the all versions up to 12.0. Starting version 13.0, Every version of the module is sold separately.
Is there any dark mode option in the theme?
Yes, there are two modes: Dark mode and Light mode. Users can select from their preferred mode.
Do you provide User Access Management?
Yes, in the Backend theme Configuration, you can provide access to a user like Settings, Advance Settings or None; as per your preference.
How to set company Logo in header ?
For this, you just need to update the company logo and under “Theme Settings”, just enable the “Show Company Logo” option and it will display on the header.
Can I set Colors to a company Specific?
Yes, from the theme settings, you can set it as: User Specific or Company Specific.
